# Bookmark Manager - Fullstack Application

A modern bookmark manager built with React, TypeScript, Express, and PostgreSQL

## Tech Stack

**Frontend:**
- React + TypeScript
- Tailwind CSS
- Vite
- Axios

**Backend:**
- Node.js + Express
- TypeScript
- PostgreSQL
- JWT Authentication
- bcrypt

---

## Project Structure
```
bookmark-manager/
├── backend/     # Express API
└── frontend/    # React App
```

---

## Setup & Installation

### Backend
```bash
cd backend
npm install
# Setup PostgreSQL database (see backend/README.md)
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

---

## Features

- User Authentication (Register/Login)
- Protected Routes with JWT
- CRUD Operations for Bookmarks
- Multi-user support with data isolation
- Repsonsive UI
